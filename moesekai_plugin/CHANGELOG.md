# Changelog

## [0.0.2] - 2026-03-05

### Added
- 添加了用户账户绑定功能（bind/unbind命令）
- 添加了专门的帮助命令（dhelp）
- 支持无空格命令格式
- 支持远程浏览器配置

### Changed
- 移除了命令前的斜杠要求
- 更新了插件声明信息
- 修复了配置类型错误
- 改进了浏览器初始化错误处理
- 修改了预测数据查询的回复消息，使其更加明确

### Fixed
- 修复了浏览器初始化失败时的错误处理

## [0.0.1] - 2025-06-30

### Added
- 初始版本
- 个人信息查询功能
- 榜线预测功能
- 基本的命令处理
